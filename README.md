# red-history-timeline
长征历史时间线交互系统-大一寒假项目

##项目简介
这是一个展示长征历史的交互式时间线网页项目，包含时间轴、地图和详情面板的联动效果。

## 技术栈
- HTML5 + CSS3 + JavaScript
- Bootstrap 5 (UI框架)
- TimelineJS (时间轴库)
- 图片热区地图

## 项目结构
red-history-timeline/
├──index.html          # 主页面
├──css/               # 样式文件
│└── style.css
├──js/               # JavaScript文件
│└── main.js
├──data/             # 数据文件
│└── events.json
├──images/           # 图片资源
│├── map/         # 地图图片
│└── events/      # 事件图片
└──README.md        # 项目说明
